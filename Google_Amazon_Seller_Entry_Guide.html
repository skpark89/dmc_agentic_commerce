⚠ 패션 카테고리의 함정

패션은 판매 수수료가 17%로 가장 높고, 반품률도 20~30%에 달합니다. 반품 시 FBA 반품 처리 수수료가 추가로 붙고, 상품 상태에 따라 재판매 불가 → 폐기 비용까지 발생합니다. 실질 마진이 10% 미만이 되는 경우가 빈번합니다.

5 AI 추천 vs 에이전틱 커머스 비교

기존 AI 추천과 에이전틱 커머스는 근본적으로 다릅니다

🤖 핵심 구분: "추천해주는 AI" vs "대신 사주는 AI"

AI 추천은 "이 상품이 좋을 것 같아요"라고 알려주고, 구매는 사람이 직접 합니다. 기존 Google Shopping, Amazon 추천 엔진이 이것입니다.
에이전틱 커머스는 "이 상품을 주문해뒀어요"까지 AI가 수행합니다. 검색→비교→장바구니→결제→주문 확인을 에이전트가 자율적으로 완료합니다.

비교 항목 기존 AI 추천 에이전틱 커머스
AI의 역할"이 상품 어때요?" → 추천까지만"주문 완료했습니다" → 결제까지 완료
사용자 행동검색 → 클릭 → 비교 → 장바구니 → 결제 (5~10단계)"이거 사줘" 한마디 (1단계)
결제 주체소비자가 직접 결제AI 에이전트가 토큰화된 결제 수단으로 대리 결제
데이터 형식키워드 매칭 + 협업 필터링구조화된 피드 (ACP/UCP) + 대화형 속성
개인화 수준구매 이력 기반 (과거 행동)대화 맥락 기반 (현재 의도 + 제약조건)
판매자 필요 역량SEO, 키워드 최적화, 광고 입찰데이터 품질, 대화형 속성, 프로토콜 준수
경쟁 기준누가 광고비를 더 쓰는가 (입찰 경쟁)누구의 데이터가 더 정확하고 풍부한가 (품질 경쟁)
전환 퍼널긴 퍼널 (이탈률 높음)초단축 퍼널 (대화 → 결제)

🟦 Google: 기존 Shopping vs AI Mode

기존: Google Shopping (AI 추천)

작동 방식키워드 검색 → 상품 카드 노출 → 클릭 → 판매자 사이트로 이동 → 판매자 사이트에서 결제
AI 역할검색어와 상품 데이터 매칭, 관련성 순위 결정
결제 장소판매자 웹사이트
전환율무료 리스팅 2.07%, 유료 광고 1.70%
판매자 노력피드 업로드 + 키워드 최적화 + 광고 입찰

신규: AI Mode Native Commerce (에이전틱)

작동 방식자연어 대화 → Gemini가 맥락 이해 → 상품 추천 + 비교 → 대화 안에서 Google Pay로 결제 완료
AI 역할의도 파악 → 상품 탐색 → 비교 분석 → 결제 실행까지 전 과정
결제 장소Google AI Mode 안 (판매자 사이트 방문 불필요)
전환율AI 트래픽 전환 품질이 기존 대비 현저히 높음 (+393% YoY 트래픽 성장)
판매자 노력피드 품질 최적화 + UCP 매니페스트 + 대화형 속성

💡 Google의 변화 핵심

"입찰 경쟁에서 데이터 품질 경쟁으로" 전환되고 있습니다. AI Mode에서는 피드 품질이 노출을 결정하지, 광고 입찰 금액이 결정하지 않습니다. 기존 검색에서 잘 나오던 상품이 AI Mode에서는 완전히 보이지 않을 수 있고, 반대로 데이터가 풍부한 상품은 광고 없이도 AI가 적극 추천합니다.

🟧 Amazon: 기존 검색 vs Alexa for Shopping (구 Rufus)

📢 2026년 5월 최신: Rufus → Alexa for Shopping 전환

Amazon은 2026년 5월, Rufus 챗봇을 폐지하고 Alexa for Shopping으로 전략을 전환했습니다. Rufus의 상품 추천 AI와 Alexa+의 에이전트 기능을 통합하여, 단순 추천을 넘어 사용자 대신 주문까지 완료하는 풀 에이전틱 쇼핑을 목표로 합니다.

기존: Amazon 검색 + Rufus (AI 추천)

작동 방식키워드 검색 → A9/COSMO 알고리즘이 순위 결정 → 클릭 → Amazon 내 결제
Rufus 역할대화형 Q&A + 상품 비교 추천 (결제는 사용자가 직접)
개인화구매이력·검색이력 기반 — 같은 질문도 계정에 따라 다른 추천
전환 효과Rufus 사용자는 비사용자 대비 전환율 60% 높음
수수료 영향일반 Amazon 수수료와 동일 (추가 과금 없음)

신규: Alexa for Shopping (에이전틱)

작동 방식"알렉사, 지난번 산 이어폰 다시 주문해줘" → 이력 확인 → 가격/재고 확인 → 자동 결제·주문
Alexa 역할의도 파악 → 상품 탐색 → 비교 → 결제 실행까지 자율 수행
개인화Alexa+ 개인 데이터 (IoT, 음성 이력, 스마트홈) + 구매이력 결합
전환 효과퍼널 제거 → 극단적 전환율 향상 기대 (데이터 미공개)
수수료 영향예상: 일반 수수료와 동일 + 프리미엄 배치 광고 상품 가능성

💡 Amazon의 변화 핵심

"Amazon 안에서의 에이전틱 커머스"입니다. Google이 열린 웹에서 에이전트를 만드는 반면, Amazon은 자기 플랫폼 안에서 에이전트를 강화합니다. 셀러 입장에서는 Amazon 밖으로 나가는 것이 아니라, Amazon 안에서 AI가 더 똑똑해지는 것이므로, 기존 Amazon 셀러라면 추가 투자 없이 자연스럽게 에이전틱 커머스에 진입하게 됩니다. 다만 AI가 추천하는 기준이 키워드 매칭에서 상품 데이터의 깊이와 정확성으로 이동하므로, A+ 콘텐츠, 풍부한 Q&A, 정확한 속성 정보가 더 중요해집니다.

✅ 결론: "둘 다 하되, 역할을 나눠라"

최적의 전략은 양쪽 모두 진입하되 역할을 분리하는 것입니다.

Google = 브랜드 구축 채널 — 자체 사이트로 트래픽을 유도하고, 고객 데이터를 직접 확보하며, 마진을 지킵니다. AI Mode가 확산되면 UCP로 대화 내 직접 판매까지 가능해집니다.

Amazon = 볼륨 채널 — 이미 구축된 3억 고객 트래픽과 물류 인프라를 활용해 판매량을 확보합니다. 마진은 낮지만 규모로 보상합니다.

그리고 두 채널 모두에서 핵심은 같습니다: "상품 데이터 품질이 곧 AI 시대의 SEO". 마케팅 문구를 빼고, 사실적으로 풍부하게, 구조화된 형태로 상품 정보를 정비하는 것이 모든 AI 플랫폼에서의 성공 기반입니다.
