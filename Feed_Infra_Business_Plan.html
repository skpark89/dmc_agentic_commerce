"Korean Commerce Feed Transformer" — 피드 인프라 SaaS 사업의 구체적 청사진

📋 Executive Summary

글로벌 에이전틱 AI 시장은 2026년 약 $9.14B 규모로, Fortune Business Insights 기준 CAGR 40.5%로 2034년 $139.19B까지 성장 전망입니다(Mordor Intelligence: 리테일/이커머스 부문만 2031년 $218.37B). 2026년 3월 Google UCP가 Cart·Catalog·Identity Linking 기능을 추가하고, 4월 OpenAI ACP가 오픈소스화(Agentic AI Foundation 거버넌스)되면서 듀얼 프로토콜(UCP+ACP) 피드 관리가 필수가 되었습니다. Syndigo는 Synapse(에이전틱 PXM)를, Akeneo는 Adaptive Product Cloud·PX Insights를 출시하며 시장을 선점 중이나, 한국어·한국 플랫폼(네이버·카페24·고도몰)을 지원하는 솔루션은 전무합니다. 한국에서는 네이버 AI 쇼핑 에이전트 출시(2026.02), 카카오 카나나 커머스 연동, 과기정통부 에이전틱 AI 얼라이언스(250+개 기관) 출범 등으로 에이전틱 커머스 전환이 가속화되고 있습니다. 한국 이커머스 판매자 약 60만+가 AI 에이전트 시대에 대비해야 하는 상황에서, 이 시장 공백을 메우는 피드 인프라 SaaS는 연 100억원+ 매출 잠재력을 가진 사업 기회입니다.

1. 핵심 기능 요소

Korean Commerce Feed Transformer의 6대 핵심 기능 모듈

CORE

① 피드 수집 & 커넥터

네이버 커머스 API, 카페24 REST API, 고도몰/메이크샵 API, 쿠팡 Wing API에서 상품 데이터를 실시간 수집. 플랫폼별 전용 커넥터로 상품명·가격·재고·옵션·이미지·리뷰를 통합 수집. EP(Exchange Protocol) 피드와 Open API 양쪽 모두 지원.

CORE

② AI 속성 변환 엔진

한국어 상품 설명에서 대화형 속성(Conversational Attributes)을 자동 추출. NLP 파이프라인이 "가볍고 출장갈 때 좋은" → {use_case: "business_travel", weight: "light"}로 변환. 한국어 형태소 분석 + LLM 조합. 카테고리별 속성 템플릿 관리.

CORE

③ 듀얼 프로토콜 피드 생성기

ACP 호환 JSON (OpenAI 오픈소스 규격, Agentic AI Foundation 거버넌스), UCP .well-known/ucp 엔드포인트 (2026.03 Cart·Catalog·Identity Linking 확장), Schema.org JSON-LD 마크업을 자동 생성. Instant Checkout 종료(2026.03) 이후 가맹점 자체 체크아웃이 필수가 되면서 듀얼 프로토콜(UCP+ACP) 피드의 별도 관리가 핵심 과제. 하나의 원본 데이터에서 각 플랫폼 규격에 맞는 피드를 동시 발행.

ADVANCED

④ MCP Server 호스팅

판매자별 전용 MCP Server를 자동 생성·호스팅. AI 모델(Claude, ChatGPT 등)이 직접 상품 데이터를 쿼리할 수 있는 엔드포인트 제공. Anthropic MCP 규격 완전 준수. 카카오 PlayMCP 호환.

ADVANCED

⑤ 한국 인증 메타데이터

KC인증·식약처인증·유기농인증·전기용품안전인증 등 한국 필수 인증 정보를 구조화된 메타데이터로 변환. AI 에이전트가 "이 제품 KC인증 받았어?" 같은 질문에 정확히 답할 수 있도록. 인증 만료일 알림 기능 포함.

VALUE-ADD

⑥ AI 노출 분석 대시보드

내 상품이 ChatGPT·Google AI Mode·네이버 AI에서 얼마나 추천되는지 기본 분석. 피드 품질 점수(Feed Health Score), 누락 속성 알림, 경쟁사 대비 AI 노출률 비교. 피드 개선 → 노출 증가 연결 증명.

4. 차별화 요소

글로벌 경쟁사 및 국내 잠재 경쟁사 대비 핵심 차별점

🌐 vs 글로벌 (Syndigo Synapse, Akeneo Adaptive Cloud)

  • ✅ 한국어 NLP 특화 — 한국어 상품 설명의 뉘앙스, 축약어, 비정형 텍스트 처리. Syndigo Synapse(2026.03 출시)와 Akeneo PX Insights 모두 영어 NLP 중심으로 한국어 미지원
  • ✅ 한국 플랫폼 네이티브 커넥터 — 네이버 EP/API, 카페24, 고도몰, 메이크샵, 쿠팡 Wing 직접 연동. Syndigo SynapseGo(2026.04)는 대화형 워크플로우 제공하나 한국 플랫폼 미지원
  • ✅ 한국 인증 메타데이터 — KC, 식약처, 유기농 등 한국 필수 인증. Akeneo Adaptive Product Cloud의 Responsive Catalog에도 한국 규제 데이터 미포함
  • ✅ 가격 경쟁력 — Syndigo/Akeneo 엔터프라이즈 요금(연 $50K+) 대비 1/10 가격으로 한국 중소 판매자 접근 가능
  • ✅ 네이버 AI 쇼핑 에이전트 최적화 — 네이버가 2026.02 출시한 AI 쇼핑 에이전트(상품 요약·비교·리뷰 분석)에 최적화된 피드 제공. 글로벌 솔루션은 접근 불가
  • ✅ 카카오 카나나 커머스 연동 — 카카오톡 선물하기·무신사·올리브영 연동 에이전트 커머스 실험(2026.04~)에 대응 가능한 유일한 한국 피드 인프라

🇰🇷 vs 국내 잠재 경쟁사

  • ✅ 듀얼 프로토콜(UCP+ACP) 동시 관리 — Microsoft가 UCP와 ACP 모두 필수 채택(2026.04)하면서 듀얼 프로토콜 피드 관리가 핵심. ACP + UCP + MCP + Schema.org를 하나의 원본에서 동시 생성. 국내에는 이 조합을 이해하는 팀 부재
  • ✅ AI 속성 추출 기술력 — 단순 피드 변환이 아닌 LLM 기반 대화형 속성 자동 생성. SEO 도구와의 근본적 차이
  • ✅ MCP Server 자동 호스팅 — 판매자별 MCP Server를 원클릭으로 배포. 카카오 PlayMCP 생태계와 즉시 호환
  • ✅ 에이전틱 AI 얼라이언스 연계 — 과기정통부 주도 에이전틱 AI 얼라이언스(250+개 기관, 2026.04 출범)의 산업·기술·생태계 분과와 연계 가능. 표준화 논의에 조기 참여
  • ✅ 글로벌 프로토콜 파트너십 — OpenAI ACP(Agentic AI Foundation 거버넌스), Google UCP 얼리 파트너로 최신 규격 변화에 즉시 대응 가능
  • ✅ First Mover — 2026년 5월 현재 한국에 동일 포지션 경쟁자 없음. 12-18개월의 선점 기회
