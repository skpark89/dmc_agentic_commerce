판매자가 진짜 준비해야 할 것
UCP & ACP 실전 가이드

복잡한 프로토콜 이론은 잊으세요. 판매자에게 필요한 것은 딱 두 가지 — Google UCP(AI Mode/Gemini)와 OpenAI ACP(ChatGPT)입니다.

⚡ 핵심 진실: 에이전트 커머스의 미래는 어떤 플랫폼을 선택하느냐가 아니라, 상품 데이터의 성숙도에 달려 있습니다.
UCP와 ACP 모두 동일한 전략적 진실로 수렴합니다.
UCP vs ACP — 판매자 관점 비교
두 프로토콜 모두 준비하는 것이 이상적이지만, 각각의 특성을 이해해야 합니다
항목 🔵 Google UCP 🟣 OpenAI ACP
노출 채널 Google AI Mode, Gemini 앱, Business Agent ChatGPT (Instant Checkout)
데이터 소스 Google Merchant Center 피드 전용 피드 엔드포인트 (JSONL/CSV/XML)
결제 처리 Google Pay + AP2 Mandate Stripe Shared Payment Token (SPT)
수수료 현재 미공개 (파일럿) 4% (OpenAI) + ~2.9%+$0.30 (Stripe)
가입 방법 Merchant Center → UCP 웨이트리스트 chatgpt.com/merchants 신청
Shopify 연동 자동 (UCP Agent 앱 설치) 자동 (Shopify Admin에서 활성화)
자체몰 연동 API + .well-known/ucp 엔드포인트 5개 REST API + 피드 엔드포인트 구현
현재 상태 Etsy, Wayfair 라이브 / 확대 중 미국 Shopify 판매자 100만+ 연동
성과 데이터 에이전트 트래픽 전환율 +28% (기존 검색 대비) 양쪽 동시 구현 시 에이전트 트래픽 +40%
⓪ 최우선: 상품 데이터 품질 (공통)
UCP든 ACP든, 모든 것의 시작점은 상품 데이터입니다. 이것이 없으면 아무것도 시작할 수 없습니다.

피드 품질 = AI 추천 품질 = 매출

PMax 광고 지출의 74~97%가 피드 기반 | 피드 오류 → AI 에이전트가 상품을 자동 제외

📋 상품 데이터 속성 체크리스트

등급 속성 설명 AI 에이전트가 활용하는 방식
필수 title 상품명 (UCP: 제한없음 / ACP: 150자) 자연어 검색 매칭, 추천 문장 생성
필수 description 상세 설명 (ACP: 5,000자) 사용자 질문에 답변, 비교 분석 근거
필수 price / priceCurrency 가격 + 통화코드 (ISO 4217) 가격 비교, 예산 필터링
필수 availability 재고 상태 (실시간 동기화 필수) 구매 가능 여부 즉시 판단
필수 image_link + 추가 이미지 3장+ 메인 이미지 + 다양한 각도 비전 모델이 색상, 소재, 스타일, 사용 맥락 분석
필수 GTIN / SKU / brand 글로벌 상품 식별자 Shopping Graph 매칭, 정품 확인
필수 shipping / return_policy 배송 조건, 반품 정책 UCP 체크아웃에 필수 (MerchantReturnPolicy)
▼ 아래는 에이전트 시대 신규 속성 — 경쟁사 대비 차별화 핵심
신규 product_highlight 상품 핵심 차별화 포인트 (예: "IPX7 방수", "100% 유기농") AI가 "왜 이 상품을 추천하는지" 설명할 때 활용
신규 product_detail (FAQ) 자주 묻는 질문과 답변 대화형 쇼핑에서 사용자 질문에 직접 답변
신규 compatible_products 호환 액세서리·부속품 "이 카메라에 맞는 렌즈는?" 같은 질문에 연계 추천
신규 substitute_products 대체 가능 상품 품절·예산 초과 시 대안 자동 추천
신규 use_scenario 사용 시나리오 (예: "캠핑", "출퇴근") "캠핑용 텐트" 같은 목적 기반 검색에 매칭
권장 native_commerce = true UCP 체크아웃 활성화 플래그 이 값이 false이면 AI Mode 체크아웃 대상에서 제외됨

⚠️ 데이터 불일치 = 즉시 제외

상품 페이지(PDP)에 49,900원인데 피드에 59,900원이면 → AI 에이전트가 "신뢰할 수 없는 데이터"로 판단하여 해당 상품을 자동 제외합니다. 웹사이트, Merchant Center 피드, Schema.org 마크업 간 가격·재고·배송 정보가 100% 일치해야 합니다.

① Google UCP 준비 단계
Google AI Mode와 Gemini에 내 상품이 노출·판매되기 위한 준비
1

Merchant Center 계정 정비

이미 Merchant Center를 사용 중이라면 기존 계정을 활용. 없다면 새로 생성.

✅
상품 피드 오류 0건 상태 달성 (disapproval 모두 해결)
✅
배송(shipping) 및 반품(return) 정책 설정 완료
✅
신규 데이터 속성 (product_highlight, FAQ, 호환제품 등) 추가
✅
native_commerce = true 설정 (체크아웃 대상 상품)
2

결제 인프라 준비

Google Pay & Wallet Console 계정 생성 + PSP(결제 서비스 제공자) 연동.

💳
Google Pay API 연동 (PSP가 미연동이면 PSP에 온보딩 요청)
🔐
AP2 Mandate 기반 결제 지원 확인 (Visa, Mastercard 등)
3

UCP 통합 (플랫폼별 방법)

Shopify: App Store에서 "Universal Commerce Agent" 앱 설치 (10분 이내)

🛍️
설치 후 Agent Policy 설정: 먼저 Read-Only로 테스트 → 검증 후 Create Carts 활성화
🗂️
메타필드(Size, Color, Material 등) → UCP 표준 필드에 매핑

자체몰 (WooCommerce, Magento, 카페24 등):

🔧
.well-known/ucp 엔드포인트 구현 (AI 에이전트가 가게를 발견하는 입구)
🔧
Checkout, Identity Linking, Order Management API 구현
⚡
API 응답 속도 P99 < 100ms 목표 (느리면 AI가 다른 판매자 선택)
// .well-known/ucp 매니페스트 예시 { "name": "내 쇼핑몰", "capabilities": ["checkout", "catalog", "orders"], "endpoints": { "checkout": "https://mystore.com/api/ucp/checkout", "catalog": "https://mystore.com/api/ucp/catalog" }, "payment_methods": ["google_pay"], "identity_linking": true }
4

웨이트리스트 신청 & 승인

📝
developers.google.com/merchant/ucp → 웨이트리스트 등록
🔍
Google이 통합 검토 후 승인 → AI Mode / Gemini에 라이브
📊
초기 도입자: 30일 내 AI Mode 트래픽 유입, 60일 차 매출 유의미 상승
② OpenAI ACP 준비 단계
ChatGPT Instant Checkout에서 내 상품이 판매되기 위한 준비
1

판매자 신청

📝
chatgpt.com/merchants 에서 신청
🇺🇸
현재 요건: 미국 기반, Shopify 유료 플랜, Stripe Payments 활성화
🌏
글로벌 확장 예정 — 한국 판매자는 Shopify 글로벌 스토어 먼저 구축 권장
2

상품 피드 구축 (3가지 구현 필요)

① 상품 피드 엔드포인트: OpenAI 제공 엔드포인트에 매일 업데이트

📦
포맷: gzip 압축 (.jsonl.gz / .csv.gz / .xml.gz)
📋
필수 필드: title(150자), description(5,000자), price+통화, availability, images, checkout 가능 여부 플래그

② Checkout API (5개 REST 엔드포인트):

🔧
체크아웃 세션 생성/업데이트, 배송 옵션 조회, 세금 계산, 결제 처리

③ Stripe 결제 연동:

💳
Shared Payment Token (SPT) — ChatGPT가 결제 정보 노출 없이 결제 가능한 토큰
⚡
이미 Stripe 사용 중이면 코드 1줄 추가로 연동 가능
// Stripe SPT 활성화 (기존 Stripe 사용자) stripe.paymentIntents.create({ amount: 49900, currency: 'usd', payment_method_types: ['shared_payment_token'], metadata: { source: 'chatgpt_acp' } });
3

적합성 테스트 & 승인

🧪
OpenAI가 자동화된 적합성 테스트 진행: 피드 검증, 체크아웃 플로우, 결제 처리
💰
수수료: 거래당 4% (OpenAI) + ~2.9%+$0.30 (Stripe) — 약 총 7%
③ 웹사이트 구조화 데이터 (Schema.org)
피드와 별개로, 웹사이트 자체에도 AI가 읽을 수 있는 구조화 데이터가 필요합니다

필수 Schema.org 마크업

🏷️
Product — 상품 기본 정보 (name, image, description, sku, gtin, brand)
💰
Offer — 가격, 재고, 판매자, 배송, 반품 정책 (Merchant Center 피드와 100% 일치)
⭐
AggregateRating / Review — 리뷰 평점, 리뷰 수
❓
FAQPage — 상품 관련 자주 묻는 질문
📐
HowTo — 사용법, 설치 방법 (해당되는 경우)
// JSON-LD 예시 (상품 페이지에 삽입) <script type="application/ld+json"> { "@type": "Product", "name": "경량 캠핑 텐트 2인용", "image": ["main.jpg", "side.jpg", "interior.jpg", "packed.jpg"], "description": "IPX7 방수, 1.8kg 초경량, 3계절용...", "sku": "TENT-LW-2P", "brand": { "name": "캠프메이트" }, "offers": { "price": 189000, "priceCurrency": "KRW", "availability": "InStock", "shippingDetails": { ... }, "hasMerchantReturnPolicy": { ... } } } </script>
판매자 유형별 최적 경로
쉬움

Shopify 판매자가장 빠르고 쉬운 경로

🛒
UCP: App Store → "Universal Commerce Agent" 설치 → Agent Policy 설정 → 끝
🟣
ACP: Shopify Admin → ChatGPT Instant Checkout 활성화 → 끝
⏱️
소요 시간: 각 10분 이내
📋
남은 할 일: 상품 데이터 품질 향상 (신규 속성 추가), 메타필드 매핑
중급

자체몰 / 카페24 / 고도몰개발 리소스 필요

🛒
UCP: .well-known/ucp + Checkout/Catalog/Orders API 직접 구현
🟣
ACP: 피드 엔드포인트 + 5개 REST API + Stripe 연동
⏱️
소요 시간: 2~4주 (개발자 1~2명 기준)
💡
팁: UCP/ACP 연동 SaaS 솔루션이 등장하면 활용 (현재 개발 중인 스타트업 있음)
한국 특화

네이버 스마트스토어 판매자현재 직접 UCP/ACP 불가

⚠️
네이버 스마트스토어에서 직접 UCP/ACP 연동은 현재 불가
✅
당장 할 것: 네이버 쇼핑 AI 에이전트용 데이터 품질 강화 (네이버 자체 에이전트)
🌐
글로벌 진출: Shopify 글로벌 스토어를 병행 운영하여 UCP+ACP 동시 진입
📊
Google Merchant Center에 상품 피드 등록 → AI Mode 노출 준비
대형

대형 리테일러 / 브랜드풀스택 구현 + Business Agent

🛒
UCP + ACP 양쪽 모두 직접 구현 (에이전트 트래픽 +40%)
💬
Business Agent 활성화: 브랜드 전용 AI 판매원 운영
🏷️
Direct Offers 캠페인 파일럿 참여
🔗
Identity Linking (OAuth 2.0): 로열티 프로그램·멤버십 가격 연동

✅ 판매자 총정리 체크리스트

🔴 P1 = 즉시 실행   🟡 P2 = 1개월 내   🟣 P3 = 3개월 내

P1
상품 피드 오류 제로: Merchant Center 경고·오류 모두 해결
P1
가격·재고 실시간 동기화: 웹사이트 = 피드 = Schema.org 100% 일치
P1
이미지 최소 4장: 메인 + 다양한 각도/사용 맥락 3장 이상
P1
GTIN/브랜드 등록: 모든 상품에 글로벌 식별자 부여
P2
신규 속성 추가: product_highlight, FAQ, 호환제품, 대체품
P2
Schema.org 마크업: Product, Offer, FAQ, Review JSON-LD 적용
P2
배송·반품 정책 구조화: ShippingDetails, MerchantReturnPolicy 설정
P2
native_commerce = true: 체크아웃 대상 상품 플래그 활성화
P3
UCP 웨이트리스트 신청: developers.google.com/merchant/ucp
P3
ACP 판매자 신청: chatgpt.com/merchants (미국 진출 시)
P3
Business Agent 활성화: Merchant Center에서 설정 (미국 대상)
P3
Direct Offers 파일럿 신청: Google Ads에서 오퍼 전략 설정

📌 판매자를 위한 한 줄 결론

지금 당장 Merchant Center 상품 피드를 완벽하게 만드세요.
가격·재고 실시간 동기화, 이미지 4장 이상, 신규 속성(FAQ, 호환제품) 추가 — 이것만 해도 UCP와 ACP 양쪽 모두에 대비됩니다.
