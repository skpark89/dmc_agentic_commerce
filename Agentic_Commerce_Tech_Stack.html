프로토콜 발전 타임라인
2024년 11월 ~ 2026년 현재까지 주요 이벤트
2024년 11월

Anthropic, MCP 발표

AI와 외부 도구 연결의 오픈 표준. AI 업계 최초의 본격적 에이전트 인프라 프로토콜.

2025년 4월

Google, A2A 프로토콜 출시

에이전트 간 통신 표준. 50+ 기술 기업 파트너 동시 참여. 6월 Linux Foundation에 기부.

2025년 5월

Coinbase, x402 출시 + ANP, W3C 백서

HTTP 402 기반 기계 간 암호화폐 결제 프로토콜 등장. 동시에 ANP가 W3C 백서로 에이전트 네트워크 표준 제안.

2025년 9월

AP2 + ACP 동시 발표

Google이 60+ 파트너와 AP2 결제 프로토콜 발표. OpenAI+Stripe가 ACP 발표, ChatGPT에서 Instant Checkout 출시.

2025년 12월

Anthropic, MCP를 Linux Foundation에 기부

Anthropic+Block+OpenAI가 공동으로 Agentic AI Foundation(AAIF) 설립. MCP 거버넌스를 중립 기관으로 이관.

2026년 1월

Google, UCP 발표

Shopify, Walmart, Target 등과 함께 풀스택 커머스 프로토콜 발표. MCP+A2A+AP2 내장 호환.

2026년 3월

UCP 업데이트 + 신한카드 실거래

UCP에 Cart, Catalog, Identity Linking 기능 추가. 한국에서 신한카드가 마스터카드와 AI 에이전트 페이 실거래 성공.

2026년 현재

스택 수렴 단계

MCP(도구) + A2A(에이전트 통신) + UCP/ACP(커머스) + AP2/x402(결제)로 역할 분담 확립. 실전 도입 가속화.

향후 전망 & 시사점
표준 경쟁

양대 진영 공존 → 점진적 수렴

Google(UCP) vs OpenAI(ACP) 구도지만, MCP와 A2A는 이미 양측 공통 채택. 판매자는 당분간 양쪽 모두 지원 필요하나, 장기적으로 상호운용성 합의가 진행될 전망.

결제 혁신

전통 결제 + 암호화폐 결제 병행

AP2(카드/은행)와 x402(스테이블코인)가 공존하며, Google AP2는 이미 x402와 통합(A2A x402). 마이크로결제와 기계 간 결제에서 암호화폐가 강점.

거버넌스

Linux Foundation 중심 중립화

MCP, A2A 모두 Linux Foundation으로 이관 완료. 특정 기업 독점 방지, 오픈소스 중심 발전. AAIF(Anthropic+Block+OpenAI)가 핵심 거버넌스 기구.

한국 기회

한국 기업의 빠른 대응이 관건

신한카드(AP2+마스터카드), 네이버(쇼핑 에이전트), 과기정통부(에이전틱 AI 얼라이언스)가 이미 움직이는 중. 글로벌 표준 참여와 국내 생태계 구축 병행 필요.
