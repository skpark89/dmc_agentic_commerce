Syndigo & Akeneo
AI 시대 상품 데이터의 핵심을 쉽게 이해하기

"AI 에이전트가 내 상품을 추천하려면, 먼저 내 상품을 이해해야 합니다."
이 두 회사가 정확히 무슨 일을 하는지, 왜 중요한지, 그리고 왜 한국에는 아직 없는지를 쉽게 설명합니다.

🧩 먼저 이해해야 할 것: "상품 데이터"가 왜 중요해졌나?

비유로 설명하겠습니다.

백화점에 가면 직원이 "어떤 걸 찾으세요?"라고 물어봅니다. 직원은 매장의 모든 상품을 알고 있기 때문에 "가볍고 출장갈 때 좋은 가방"이라고 말하면 적절한 상품을 찾아줍니다.

이제 AI 에이전트가 그 직원 역할을 합니다. ChatGPT에게 "출장용 가방 추천해줘"라고 말하면 AI가 여러 쇼핑몰의 상품 중 최적의 것을 골라줍니다.

그런데 문제가 있습니다. 쇼핑몰의 상품 데이터는 원래 "사람이 보는 웹페이지"를 위해 만든 것입니다. 예쁜 이미지, HTML로 된 상세페이지, 키워드 나열... AI가 이걸 "이해"하기 어렵습니다.

❌ 기존 상품 데이터 (사람용)

예쁜 이미지, 긴 HTML 상세페이지, 키워드 나열, 비정형 텍스트.
AI가 읽으면 핵심 정보를 파악하기 어려움.

상품명: [특가] 가죽 서류가방 블랙 A-302
상세: <img src="..."> <div>고급 소가죽...
<b>사이즈: 가로39 세로28</b>...
카테고리: 패션 > 가방 > 서류가방
→

✅ AI-Ready 상품 데이터 (AI용)

구조화된 JSON, 대화형 속성, 인증 정보, 리뷰 요약.
AI가 즉시 이해하고 고객에게 자연스럽게 설명 가능.

{
  "title": "소가죽 서류가방",
  "use_case": ["출장", "비즈니스"],
  "weight": "680g (가벼운)",
  "material": "소가죽",
  "laptop_fit": "15인치까지",
  "reviews_summary": "가볍고 튼튼...",
  "faq": [{"Q":"방수되나요?","A":"..."}]
}

이 변환을 해주는 회사가 바로 Syndigo와 Akeneo입니다.
이들은 "사람용 상품 데이터"를 "AI가 이해하는 상품 데이터"로 바꿔주고, 그 데이터를 ChatGPT, Google AI, Claude 같은 AI 에이전트에 직접 연결해줍니다.

"대화형 속성(Conversational Attributes)"이란?

AI 에이전트가 고객과 자연스러운 대화를 나누기 위해 필요한 새로운 형태의 상품 정보

💬 쉽게 말하면

기존 상품 데이터는 "사이즈: 39×28cm, 무게: 680g, 색상: 블랙" 같은 스펙 나열이었습니다.
대화형 속성은 "이 가방은 가볍고, 15인치 노트북이 들어가고, 출장갈 때 딱 좋아요"처럼 AI가 고객과 대화할 때 사용할 수 있는 맥락이 있는 정보입니다.

⚡ 왜 이게 중요한가?

Google이 Merchant Center에 대화형 속성(Conversational Attributes)을 새 필드로 추가했고, OpenAI의 ACP도 이런 구조화된 데이터를 요구합니다.
대화형 속성이 없는 상품은 AI 에이전트에게 "보이지 않는 상품"이 됩니다. 마치 SEO를 안 하면 구글 검색에 안 나오는 것처럼, 대화형 속성이 없으면 AI 추천에서 빠집니다.

🏢 이 회사가 뭐 하는 회사인가?

한 줄 요약: 브랜드/제조사의 상품 정보를 한곳에 모아서 관리하고, 이것을 아마존·월마트·쿠팡 같은 유통 채널에 자동으로 배포해주는 상품 정보 관리(PIM) 플랫폼입니다.

비유: 마치 "상품 정보의 우체국"과 같습니다. 브랜드가 상품 정보(이름, 사진, 스펙, 가격 등)를 Syndigo에 넣으면, Syndigo가 알아서 아마존용, 월마트용, 이제는 ChatGPT용으로 포맷을 바꿔서 보내줍니다.

미국 상위 20개 리테일러 중 90%가 Syndigo를 사용합니다.

🚀 서비스 ③: Synapse™ — 업계 최초 에이전틱 PXM 플랫폼 2026.03 NEW

한 줄 요약: 2026년 3월 출시된 Synapse는 엔터프라이즈급 MDM(마스터 데이터 관리) 위에 AI 에이전트를 네이티브로 탑재한 업계 최초의 에이전틱 PXM 플랫폼입니다.

핵심 개념: 기존 PIM은 "사람이 데이터를 정리하는 도구"였다면, Synapse는 "AI 에이전트가 자율적으로 상품 데이터를 관리하는 시스템"입니다. AI 에이전트가 상품 생성·온보딩·콘텐츠 보강·유통 배포까지 전 과정을 자율 수행하되, 사람이 감독(Human Oversight)합니다.

주요 에이전트:
• Product Creation Agent — 신상품 등록을 자동으로 가속
• Onboarding Agent — 공급업체·상품 온보딩을 간소화
• Enrichment Agent — 대규모 콘텐츠 완성도·품질 향상
• Syndication Agent — 리테일러·마켓플레이스별 검증 후 자동 배포

상품·공급업체·위치·컴플라이언스·성과 데이터를 "디지털 트윈"으로 연결하여, AI 에이전트가 전체 맥락을 이해한 상태에서 작업합니다.

💬 서비스 ④: SynapseGo™ — 대화형 에이전틱 인터페이스 2026.04 NEW

한 줄 요약: 2026년 4월 Syndigo Connect 컨퍼런스에서 공개된 SynapseGo는 자연어 대화만으로 에이전틱 워크플로를 실행할 수 있는 인터페이스입니다.

비유: Synapse가 "AI 에이전트 군단"이라면, SynapseGo는 "그 군단에게 말로 지시하는 리모컨"입니다. 기술 전문가가 아닌 마케팅·MD 팀도 채팅으로 "이번 시즌 신상품 200개 등록해줘"라고 요청하면 AI 에이전트가 자동 실행합니다.

핵심 포인트: 엔터프라이즈급 거버넌스(정책 적용, 검증, 감사 추적)가 내장되어 있어, AI가 자율적으로 일하되 브랜드의 통제력은 유지됩니다.

🌐 서비스 ⑤: UCP(Universal Commerce Protocol) 지원

Syndigo는 OpenAI ACP뿐 아니라 Google이 주도하는 UCP(Universal Commerce Protocol)도 적극 지원합니다. UCP는 Shopify, Target, Walmart 등 20개 이상의 글로벌 파트너가 참여하는 오픈소스 표준으로, AI 에이전트의 상품 발견→평가→구매→이행 전 과정을 커버합니다.

Syndigo의 GEO와 OpenAI Connect가 ACP·UCP 양쪽 모두를 지원함으로써, 브랜드는 ChatGPT든 Google AI든 하나의 파이프라인으로 대응할 수 있습니다.

비유: Syndigo가 "상품 정보의 우체국"이라면, Akeneo는 "상품 정보의 통역사"입니다. 다양한 형태의 상품 데이터를 받아서, AI가 이해할 수 있는 언어로 번역하고, 품질을 검증하고, 안전하게 연결해줍니다.

🔌 서비스 ①: Akeneo MCP Server

한 줄 요약: AI 모델(Claude, ChatGPT 등)이 상품 데이터에 직접 접근할 수 있는 통로를 만들어줍니다.

MCP(Model Context Protocol)란 Anthropic이 만든 표준으로, "AI가 외부 데이터와 도구에 연결하는 방법"을 정한 규약입니다. 마치 USB가 어떤 장치든 컴퓨터에 연결해주는 것처럼, MCP는 AI를 외부 데이터에 연결합니다.

Akeneo MCP Server의 핵심 역할: "거버넌스(관리) 레이어"
AI가 상품 데이터에 접근할 때, 아무 데이터나 마음대로 사용하면 안 됩니다. 브랜드 가이드라인, 법적 규정, 데이터 정확성이 보장되어야 합니다. Akeneo MCP Server는 이 "교통 정리" 역할을 합니다.

💡 왜 "거버넌스"가 중요한가?

AI가 상품 정보를 마음대로 수집하면 이런 문제가 생깁니다:
① 잘못된 가격 — 옛날 가격을 최신으로 착각해서 안내
② 허위 인증 — 유기농이 아닌데 유기농이라고 추천
③ 브랜드 가이드 위반 — "저렴한 대안"으로 프리미엄 브랜드를 소개
④ 법적 위험 — 의약품 효능을 과장해서 설명

Akeneo MCP Server는 이런 문제를 데이터가 AI에 전달되기 전에 차단합니다.

🔑 이 파트너십의 핵심 포인트

역할 분담이 깔끔합니다:
• Akeneo 담당: 상품 발견(Discovery) — AI가 상품을 찾고 이해할 수 있게
• Stripe 담당: 결제(Checkout) — AI가 안전하게 결제할 수 있게

판매자 입장에서는 "Akeneo에 상품 데이터 넣고, Stripe 계정 연결하면 끝"입니다. AI 에이전트(ChatGPT 등)에서 상품이 자동으로 보이고, 고객이 사면 결제도 자동 처리됩니다.

📊 서비스 ③: PX Insights — AI Discovery Optimization 2026.04 NEW

한 줄 요약: 2026년 4월 Spring Release에서 공개된 "AI 커머스의 첫 번째 클로즈드 루프" 도구입니다. 내 상품이 ChatGPT, Gemini 등 AI 쇼핑 어시스턴트에서 어떻게 해석·추천되는지 보여주고, 데이터를 어떻게 고쳐야 하는지까지 알려줍니다.

작동 방식:
① AI 쇼핑 쿼리를 시뮬레이션 ("가벼운 출장용 노트북 추천")
② 내 상품이 AI 답변에서 몇 위로 나오는지 확인
③ 경쟁사 상품과 벤치마크 비교
④ 설명·속성 밀도를 개선할 구체적 제안을 제공
⑤ 수정 후 다시 시뮬레이션 → 순위 변화 확인

비유: SEO에 Google Search Console이 있다면, AI 시대에는 PX Insights가 "AI Search Console" 역할입니다.

🏗️ 서비스 ④: Adaptive Product Cloud — "The Great Restack" 2026.04 NEW

한 줄 요약: Akeneo가 2026년 4월 발표한 "The Great Restack" 비전으로, PIM을 단순한 "기록 시스템(System of Record)"에서 "지능 시스템(System of Intelligence)"으로 전환합니다.

핵심 변화:
• 지능형 피드백 루프 — AI 발견 트렌드, 검색 성과, 마켓플레이스 피드백을 상품 레코드에 직접 반영하여 데이터가 자동으로 개선됨
• Data Architect Agent — 리테일러 요구사항·리젝션 로그·검색 트렌드를 분석해 누락된 속성과 값을 자동 제안·생성
• Ask Ziggy — PIM 내 AI 어시스턴트로, 설정·워크플로에 대한 실시간 가이드 제공
• Vibe-Coding — 자연어와 AI 지원으로 커스텀 비즈니스 로직을 직접 개발·배포 가능

에이전틱 커머스 시대에 상품 데이터는 한 번 입력하고 끝나는 것이 아니라, 시장 신호에 따라 지속적으로 진화해야 한다는 철학입니다.

🤔 쉽게 정리하면

Syndigo = "내 상품 데이터를 ChatGPT/Google AI에 보내서 AI가 추천할 수 있게 하겠다" (Push 방식) + 2026년 Synapse로 AI 에이전트가 상품 데이터 관리 자체를 자동화
Akeneo = "AI가 필요할 때 내 상품 데이터에 와서 물어볼 수 있게 하겠다" (Pull 방식) + 2026년 PX Insights로 AI 쇼핑에서 내 상품 순위를 실시간 모니터링·최적화

둘 다 결국 "AI 에이전트가 내 상품을 정확히 이해하고 추천할 수 있게 만드는 것"이 목적입니다. 접근 방식만 다르지만, 2026년에는 양사 모두 "에이전틱 PXM" — AI 에이전트가 상품 데이터를 자율적으로 관리·최적화하는 방향으로 수렴하고 있습니다.

왜 한국에는 이런 도구가 없고, 왜 필요한가?

❌ 한국 플랫폼 미지원

Syndigo와 Akeneo는 Shopify, Amazon, WooCommerce 같은 글로벌 플랫폼만 지원합니다. 네이버 스마트스토어, 카페24, 고도몰, 메이크샵은 연결 자체가 불가능합니다.

❌ 한국어 처리 불가

"가성비 좋은", "출장갈 때 딱인", "이건 좀 그런데 그래도 괜찮아요" 같은 한국어 표현에서 대화형 속성을 추출하는 NLP 기능이 없습니다. 영어 기반으로만 최적화되어 있습니다.

❌ 한국 인증 체계 미지원

KC인증, 식약처인증, 유기농인증 등 한국 고유 인증마크를 AI가 이해할 수 있는 구조화 데이터로 변환하는 기능이 없습니다. 글로벌 스키마에 한국 인증 필드 자체가 없습니다.

⚠️ 한국 판매자가 처한 상황 — 2026년 5월 현재

지금 한국 판매자 60만+는 다음과 같은 상황에 놓여 있습니다:

  • 📌 ChatGPT Shopping이 확장 중인데, 내 상품이 추천되려면 ACP 호환 피드가 필요. 하지만 네이버 스마트스토어 EP를 ACP JSON으로 바꿔주는 도구가 없음.
  • 📌 Google AI Mode가 쇼핑을 혁신 중인데, UCP .well-known 엔드포인트를 자동 생성해주는 도구가 없음.
  • 📌 Claude, Perplexity도 쇼핑 기능을 도입 중인데, MCP Server를 통해 내 상품 데이터를 연결해주는 도구가 없음.
  • 📌 카페24가 ChatGPT Apps에 진출했지만, 개별 판매자가 자기 상품을 AI-Ready로 최적화하는 셀프서비스 도구가 없음.
  • 🔥 네이버가 2026년 상반기 AI 쇼핑 에이전트(Agent N)를 출시 — 자연어로 상품 검색·비교·구매까지 처리. 뷰티·식품 카테고리 우선 적용, 실시간 트렌드 분석·장바구니 기능 확장 중. 네이버 내 판매자 상품이 AI 에이전트에 추천되려면 구조화된 대화형 속성이 필수.
  • 🔥 카카오가 KakaoTalk에 AI 에이전트(Kanana) 내장 — 대화 맥락을 읽고 선제적으로 상품 추천. OpenAI 파트너십으로 ChatGPT도 카카오 서비스에 통합. 카카오톡 5,000만 사용자가 잠재 AI 쇼핑 고객으로 전환 중.
  • 📌 결론: 한국의 Syndigo/Akeneo — 네이버·카페24·고도몰 피드를 ACP/UCP/MCP 호환 대화형 속성으로 변환하고, 네이버 Agent N·카카오 Kanana·ChatGPT·Google AI에 직접 연결해주는 한국어 특화 도구가 그 어느 때보다 시급하게 필요합니다.

정리: 기억해야 할 핵심 인사이트

🎯 3줄 요약

① AI 에이전트 시대에 "상품 데이터 품질"이 곧 "매출"입니다. AI가 추천하지 않는 상품은 팔리지 않습니다. 마치 구글에 안 나오는 웹사이트처럼.

② Syndigo와 Akeneo는 2026년 "에이전틱 PXM"으로 진화했습니다. Syndigo는 Synapse/SynapseGo로 AI 에이전트가 상품 데이터를 자율 관리하고, Akeneo는 PX Insights로 AI 쇼핑 내 상품 순위를 실시간 최적화합니다.

③ 하지만 한국 시장은 여전히 빈 칸입니다. 네이버 Agent N, 카카오 Kanana 등 한국 AI 쇼핑도 빠르게 성장 중이지만, 판매자의 상품 데이터를 AI-Ready로 변환해주는 한국어 특화 PIM/PXM 도구가 전무합니다. 이것이 바로 한국 IT 기업의 기회입니다.
